The De/Coding Bootcamp

Our Teaching Philosophy Sets Us Apart

Pre-Learning

Before you learn a new technology, you will be exposed to a code base that utilizes it, so it will already feel familiar before you try to use it. In doing so, you will be harnessing your brain's largely uncounscious pattern-matching capabilities to pre-learn the material.

Keep It Simple, Dive Deep

You will build the same base project in different ways throughout the bootcamp, so you can concentrate on the new technology you're learning instead of new project details. You will get to focus on excellent implementation and big picture conding concepts, instead of irrelevant project specifications.

Portfolio Ready Projects

You will build three capstone projects of your choice from scratch, ensuring that you know each technology stack inside and out. These projects will be complex coding structures designed to set you apart from other bootcamp grads.
